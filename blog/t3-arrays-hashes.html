Arrays and Hashes

Technical Blog

Monday, July 27, 2015

Here is a basic, basic introduction to what arrays and hashes are. Please note that this is just a scratch of the surface on what these methods can further do. Enjoy!

An array is a list of items in order. For the most part, arrays are indexed with integers. Each slot inbetween the brackets (see examply below) will act as a vairable. Each variable can only hold one item at a time, but an array can hold mulitple items called elements of an array.

An example of an array: ![Array](imgs/array.png)

A hash is a collection of keys and its values. It acts like a dictionary in a sense that is collects the words (in hashes case, keys) and their definition (values). Hashes are very similar to arrays execpt hashes allow you to use any object type like strings, expressions, and so on. Hashes have a default vlaue and when the value is trying to be returned to a key that does not exist, it will show a value 'nil'.
